# Cool Things I Have Done with Hermes

These are a few practical ways I have used Hermes. Once you see it in action, it is both impressive and a little scary.

- Used Hermes to update itself
  - Ask it directly:
    - `Please update Hermes`
    - `Please check for Hermes updates at 9 PM and install them if there are any`
  - You can also use the slash command: `/update`

- Used Hermes to push changes to a private repo
  - Hermes can help update documentation or code in Azure DevOps, GitHub, and other repos.

- Used Hermes to back up its config on a daily schedule

![Config backup workflow](../images/hermes-setup/H_ideas01.png)

- Used Hermes to check host performance
  - `Please check the server you're on and tell me the memory and CPU usage`

- Used Hermes to troubleshoot Discord thread responses
  - Goal: keep related conversations grouped in one channel while learning.
